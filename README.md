# Interview Navigator - AI-Powered Mock Interview Platform

A production-ready React application that simulates realistic technical interviews using Google's Gemini API, with Firebase backend for persistence.

## Features

✅ **AI-Powered Questions** - Generated by Gemini 2.5 Flash with diverse topics  
✅ **Real-time Evaluation** - Instant feedback on answers with scoring (1-5)  
✅ **Interview History** - Track all sessions with metrics and analytics  
✅ **User Profiles** - Customize role, skills, and experience level  
✅ **Error Handling** - Comprehensive error management and recovery  
✅ **Type Safety** - Full TypeScript with strict mode enabled  
✅ **Testing Ready** - Vitest + React Testing Library setup  
✅ **Production Grade** - Logging, validation, retry logic, error boundaries  

## Quick Start

```bash
# Install dependencies
npm install

# Configure environment
cp .env.example .env
# Edit .env with your Firebase and Gemini API credentials

# Start dev server
npm run dev

# Build for production
npm run build
```

## Project Structure

```
src/
├── components/       # UI components (Button, Alert, ErrorBoundary)
├── pages/           # Screen components  
├── services/        # Firebase & Gemini API clients
├── hooks/           # useAuth, useInterview custom hooks
├── types/           # TypeScript definitions
├── utils/           # Logger, retry, validation, errors
└── __tests__/       # Unit and integration tests
```

## Configuration

### Firebase Setup

1. Create project at [Firebase Console](https://console.firebase.google.com)
2. Enable Authentication (Anonymous)
3. Enable Firestore Database (Test Mode)
4. Copy credentials to `.env`

### Gemini API Setup

1. Get key from [Google AI Studio](https://aistudio.google.com)
2. Add to `.env` as `VITE_GEMINI_API_KEY`

## Architecture Highlights

- **TypeScript Strict Mode** - Type-safe across entire app
- **Custom Hooks** - Encapsulated logic with React hooks
- **Error Boundaries** - Graceful error handling
- **Structured Logging** - Pino-based debugging
- **Validation** - Zod schemas for API responses
- **Retry Logic** - Exponential backoff for rate limits
- **Firestore Real-time** - Live interview history updates
