## Configuration

### Firebase Setup

1. Create project at [Firebase Console](https://console.firebase.google.com)
2. Enable Authentication (Anonymous)
3. Enable Firestore Database (Test Mode)
4. Copy credentials to `.env`

### Gemini API Setup

1. Get key from [Google AI Studio](https://aistudio.google.com)
2. Add to `.env` as `VITE_GEMINI_API_KEY`

## Architecture Highlights

- **TypeScript Strict Mode** - Type-safe across entire app
- **Custom Hooks** - Encapsulated logic with React hooks
- **Error Boundaries** - Graceful error handling
- **Structured Logging** - Pino-based debugging
- **Validation** - Zod schemas for API responses
- **Retry Logic** - Exponential backoff for rate limits
- **Firestore Real-time** - Live interview history updates
